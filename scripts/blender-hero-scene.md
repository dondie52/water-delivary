# Blender hero scene (run after MCP is connected)

With Blender open and BlenderMCP connected, ask Cursor to:

1. Create a stylized water bottle with floating droplets on brand colors (#061a4f, #00a5db).
2. Add studio lighting and a slow camera orbit.
3. Render an 8-second MP4 loop and save to `public/brand/hero-loop.mp4`.

Replace the ffmpeg Ken Burns loop by re-running the site after export.

If you prefer a still fallback, render a PNG to `public/brand/hero-banner.jpg` instead.
